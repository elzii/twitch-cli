# twitch-cli

CLI to search twitch channels/games and stream them to vlc via livestreamer

### dependencies

* [livestreamer](https://github.com/chrippa/livestreamer) - `pip install livestreamer`

### install

`npm install -g twitch-cli`

### usage

`twitch`
